Explicandos as tags do HTML

Tipos de tag "meta charset".

  • utf-8: Recebe os caracteres padrões em português.
  • iso-8889: Recebe os caracteres padrões em inglês.
  • HAN-pinyn: Recebe os caracteres padrões em mandarim.

Tipos de tags estruturais

  • html lang="pt-br": Inicia a pagina HTML e também diz que a identação da pagina vai ser em pt-br, isso ocorre por conta do (lang="pt-br").
  • head: Head significa cabeça, isso quer dizer que vamos criar o cabeçalho e colocar o titulo do site e a identação de caractere.
  • header: Header significa cabeçalho, isso quer dizer que esta tag serva para colocar o cabeçalho.
  • h1,..,h7: Essas tags são usadas para colocar os titulos da pagina, sendo "h1" o maior titulo e "h7" o menor.
  • hr: Literal adiciona uma linha de cabo a rabo na pagina.
  • body: Essa tag da inicio ao corpo do texto.
  • section: Essa tag cria uma seção ao texto isso serve para encaixar melhor o texto.
  • div: A tag "div" cria um bloco genérico para agrupar conteúdos.
  • ul: A tag "ul" cria colunas para agrupação de itens genéricos.
  • li: A tag "li" serve para criar linhas para digitação de textos.
  • footer: Roda do site.
  • br: A tag "br" quebra linha.

Tipos de tags textuais.

  • title: Title significa titulo, isso que dizer que é literalmente colocar o titulo da pagina web.
  • p: A tag "p" serve para escrever textos de qualquer tipo.

Tags adicionais

  • iso-8859-15: Define a codificação da folha de estilo para Latin-9 (línguas da Europa Ocidental, com sinal de euro).
  • i: Seleciona o texto escrito e coloca italico na(s) palavra(s).
  • strong: Seleciona o texto escrito e coloca negrito na(s) palavra(s).
  • u: Seleciona o texto escrito e sublinha a(s) palavra(s).
  • img src: A tag "img src" adiciona uma imagem ao site através do link.
  • a: Tag âncora para um link.
  • abbr: Tag de abreviação de palavras.
  • address: Tag de informação de contato.
  • mark: A tag "mark" literalmente marca as palavras, tipo assim.
  • button: Cria um botão .
